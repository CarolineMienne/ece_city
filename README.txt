This project is a SimCity-like game developed in a scholar environment.
It is developed by three second year students currently (by March/April 2015)
@ ECE Paris Engineering School.

*****************************************
*         Note for contributors	 	*
*****************************************
Coding conventions are the following :
- snake_case for variables
- lowerCamelCase for functions
- all names in english
- ALL MERGES MUST BE DONE WITH THE --no-ff OPTION !!! This is for keeping clear history of branches.
